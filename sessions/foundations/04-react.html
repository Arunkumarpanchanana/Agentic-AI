ReAct: Reasoning + Acting

Learn the Thought-Action-Observation loop from the landmark 2022 paper and implement it with LangChain.

35 min Foundations Hands-on Lab 5 Quiz Questions

The ReAct Paper (Yao et al., 2022)

ReAct — short for Reasoning + Acting — was introduced by Shunyu Yao and colleagues at Princeton and Google Brain in the paper "ReAct: Synergizing Reasoning and Acting in Language Models" (2022). It became one of the most cited and practically influential papers in agentic AI.

The key insight: prior work had separated reasoning (chain-of-thought prompting) from acting (action-only agents). ReAct showed that interleaving explicit reasoning steps with action steps dramatically improved performance on knowledge-intensive and decision-making tasks.

Core idea: At each step, the LLM produces a Thought (internal reasoning trace) before producing an Action. The action's result becomes an Observation that feeds into the next Thought. This creates an interpretable, steerable loop.

The Thought-Action-Observation Loop

Here is a real ReAct trace for the question: "What is the elevation range for the area that the eastern sector of the Colorado orogeny extends into?"

Thought 1: I need to search Colorado orogeny and find the eastern sector information.
Action 1: Search["Colorado orogeny"]
Observation 1: The Colorado orogeny was an episode of mountain building (an orogeny) in Colorado... The eastern sector extends into the High Plains...
Thought 2: The eastern sector extends into the High Plains. I need to find the elevation range of the High Plains.
Action 2: Search["High Plains elevation"]
Observation 2: The High Plains are a subregion of the Great Plains. The elevation ranges from around 1,800 to 7,000 feet above sea level.
Thought 3: I now know the elevation range. The answer is 1,800 to 7,000 feet.
Action 3: Finish["1,800 to 7,000 feet"]

Notice how each Thought explains why the next action is being taken — this makes the agent's reasoning transparent and debuggable.

ReAct vs Chain-of-Thought vs Action-Only

ApproachReasoningExternal InfoError RecoveryInterpretability
Chain-of-ThoughtExplicit thoughtsNo (closed-book)PoorHigh
Action-OnlyNoneYes (tools)PoorLow
ReActExplicit thoughtsYes (tools)Good (can reason about errors)High

The paper showed ReAct outperformed chain-of-thought by 6-11% on HotpotQA (multi-hop reasoning) and significantly reduced hallucination because the agent could look up facts rather than generating them from memory.

Why Thoughts Help

The reasoning trace serves multiple roles in ReAct:

  • Decomposition: Breaking complex questions into sub-questions.
  • Context tracking: Summarising what has been learned so far.
  • Error diagnosis: "The search returned no results — I'll try a different query."
  • Goal alignment: "I only need one more piece of information before I can answer."
  • Human interpretability: Developers can read the trace to understand why an agent took a wrong turn.

LangChain ReAct Agent

from langchain import hub
from langchain.agents import AgentExecutor, create_react_agent
from langchain_openai import ChatOpenAI
from langchain.tools import Tool
from langchain_community.tools import DuckDuckGoSearchRun

# --- Define tools ---
search = DuckDuckGoSearchRun()

def calculator(expression: str) -> str:
    """Evaluate a mathematical expression."""
    try:
        result = eval(expression, {"__builtins__": {}}, {})
        return str(result)
    except Exception as e:
        return f"Error: {e}"

tools = [
    Tool(name="Search", func=search.run,
         description="Search the web for current information. Input: a search query string."),
    Tool(name="Calculator", func=calculator,
         description="Evaluate math expressions. Input: a Python-style math expression like '2 ** 10 / 3'."),
]

# --- Load the ReAct prompt from LangChain Hub ---
prompt = hub.pull("hwchase17/react")

# --- Create the ReAct agent ---
llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)
agent = create_react_agent(llm, tools, prompt)
executor = AgentExecutor(agent=agent, tools=tools, verbose=True, max_iterations=10)

# --- Run ---
result = executor.invoke({
    "input": "What is the population of Tokyo, and what is 15% of that number?"
})
print("Final answer:", result["output"])

Common ReAct Failure Modes

  • Thought-action mismatch: The thought says "I'll search X" but the action searches Y.
  • Observation ignoring: The agent generates the next thought before properly reading the observation.
  • Infinite loops: The agent keeps searching for something not on the web.
  • Premature finish: The agent answers before gathering enough information.
  • Tool hallucination: The agent calls a tool that doesn't exist in its tool list.
Debugging tip: Always set verbose=True in your AgentExecutor during development. Reading the full Thought-Action-Observation trace is the fastest way to understand why your agent is misbehaving.

Simulate ReAct Without an LLM

# Simulate a ReAct trace with a hardcoded "LLM"
def fake_react_llm(step, observation=""):
    steps = [
        ("Thought", "I need to find the GDP of Germany."),
        ("Action", "Search: GDP of Germany 2024"),
        ("Thought", f"Observation says: {observation}. Now calculate 5% of that."),
        ("Action", "Calculator: 4_456_000_000_000 * 0.05"),
        ("Thought", "I have the answer."),
        ("Action", "Finish: 5% of Germany GDP is approximately $222.8 billion"),
    ]
    return steps[step] if step < len(steps) else ("Finish", "Done")

# Run the simulated loop
print("=== ReAct Trace ===")
obs = ""
for i in range(6):
    role, content = fake_react_llm(i, obs)
    prefix = {"Thought": "💭", "Action": "⚡", "Finish": "✅"}.get(role, "")
    print(f"{prefix} {role}: {content}")
    if role == "Action" and "Search" in content:
        obs = "Germany GDP 2024: approximately $4.456 trillion (IMF estimate)"

Build the Real ReAct Agent

from langchain import hub
from langchain.agents import AgentExecutor, create_react_agent
from langchain_openai import ChatOpenAI
from langchain.tools import Tool
from langchain_community.tools import DuckDuckGoSearchRun

search = DuckDuckGoSearchRun()

tools = [
    Tool(name="Search", func=search.run,
         description="Search the internet. Input: search query string."),
    Tool(name="Calculator", func=lambda expr: str(eval(expr, {"__builtins__": {}}, {})),
         description="Evaluate math. Input: Python math expression."),
]

llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)
prompt = hub.pull("hwchase17/react")
agent = create_react_agent(llm, tools, prompt)
executor = AgentExecutor(agent=agent, tools=tools, verbose=True, max_iterations=8)

result = executor.invoke({"input": "What is the current price of gold per ounce, and how much would 5 ounces cost?"})
print("\nFinal Answer:", result["output"])
