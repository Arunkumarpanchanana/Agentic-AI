Hallucination & Reliability

Why agents hallucinate more than chatbots, and how to detect and mitigate it with self-consistency, grounding, and uncertainty estimation.

35 min Evaluation Hands-on Lab 5 Quiz Questions

Why Agents Hallucinate More Than Chatbots

A chatbot hallucinating in a single response is bad. An agent hallucinating compounds across many steps — each hallucinated fact becomes a false premise for the next step, causing error cascades. An agent that invents a citation in step 2 will build an entire research report on that false foundation by step 10.

FactorHow it amplifies hallucination in agents
Compounding errorsEach step's hallucination feeds the next step as a "fact"
Long context dilutionIn long histories, the model attends less carefully to early facts
Tool result misreadingAgent misinterprets ambiguous tool output and proceeds with wrong assumption
Goal driftAs the agent drifts from its goal, it generates content that "sounds right" but isn't grounded
No external checkUnlike a human, the agent doesn't pause and double-check surprising claims
Key insight: The solution to agent hallucination is grounding — anchoring every factual claim to an external source the agent can verify. The agent should never assert a fact it did not obtain from a tool result.

Self-Consistency Checking

Self-consistency is the most practical hallucination mitigation for agents. Sample the agent (or LLM) multiple times and compare outputs. High agreement = high confidence. High variance = likely hallucination.

from langchain_openai import ChatOpenAI
from langchain.schema import HumanMessage
from collections import Counter
import re

llm = ChatOpenAI(model="gpt-4o-mini", temperature=0.7)

def self_consistency_check(question: str, n: int = 5, threshold: float = 0.6) -> dict:
    """
    Sample the LLM n times and check if answers are consistent.
    Returns: answer, confidence, is_reliable flag.
    """
    answers = []
    for _ in range(n):
        r = llm([HumanMessage(content=f"Answer in one sentence:\n{question}")]).content.strip()
        answers.append(r)

    # Extract key claims (simplified: first number or yes/no found)
    def extract_key(answer):
        if re.search(r'\byes\b', answer.lower()): return "yes"
        if re.search(r'\bno\b', answer.lower()): return "no"
        nums = re.findall(r'\b\d[\d,\.]*\b', answer)
        return nums[0] if nums else answer[:30].lower()

    keys = [extract_key(a) for a in answers]
    counts = Counter(keys)
    top_key, top_count = counts.most_common(1)[0]
    confidence = top_count / n
    best_answer = next(a for a, k in zip(answers, keys) if k == top_key)

    return {
        "answer": best_answer,
        "confidence": confidence,
        "is_reliable": confidence >= threshold,
        "all_keys": keys,
    }

result = self_consistency_check("What year was Python first released?")
print(f"Answer: {result['answer'][:100]}")
print(f"Confidence: {result['confidence']:.0%}")
print(f"Reliable: {result['is_reliable']}")

Factuality Grounding

Grounding means every factual claim must be traceable to a retrieved source, not generated from model weights:

def grounded_response(question: str, search_fn) -> dict:
    """Force agent to ground every claim in retrieved evidence."""
    # Step 1: Retrieve evidence
    evidence = search_fn(question)
    # Step 2: Generate grounded response
    grounded_llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)
    response = grounded_llm([HumanMessage(content=f"""Answer the question using ONLY the provided evidence.
If the evidence doesn't contain the answer, say "I don't have sufficient evidence to answer this."
Do NOT use any knowledge from your training data.

Evidence:
{evidence}

Question: {question}
Answer:""")]).content
    return {"answer": response, "evidence": evidence, "grounded": True}

Uncertainty Estimation

Instead of always giving a confident answer, agents should express calibrated uncertainty:

UNCERTAINTY_PROMPT = """Answer the question, then rate your confidence:
- HIGH: I'm certain based on well-established facts
- MEDIUM: I believe this is correct but you should verify
- LOW: I'm uncertain — please verify this independently

Question: {question}

Answer: [your answer]
Confidence: [HIGH/MEDIUM/LOW]
Reason for confidence level: [brief explanation]"""

def answer_with_uncertainty(question: str) -> dict:
    llm_det = ChatOpenAI(model="gpt-4o-mini", temperature=0)
    response = llm_det([HumanMessage(content=UNCERTAINTY_PROMPT.format(question=question))]).content
    # Parse confidence level
    confidence = "UNKNOWN"
    for level in ["HIGH", "MEDIUM", "LOW"]:
        if level in response:
            confidence = level
            break
    return {"response": response, "confidence_level": confidence,
            "should_verify": confidence in ["LOW", "UNKNOWN"]}
Hallucination detection pipeline: (1) Ground all factual claims in tool results. (2) Use self-consistency to flag low-confidence answers. (3) Add a dedicated verifier agent that checks key claims against retrieved evidence before passing results downstream. Don't rely on a single LLM call for anything factual.

Build a Claim Extractor

from langchain_openai import ChatOpenAI
from langchain.schema import HumanMessage
import re

llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)

def extract_claims(text: str) -> list:
    """Extract verifiable factual claims from text."""
    response = llm([HumanMessage(content=f"""Extract all verifiable factual claims from this text.
Return as a numbered list. Only include specific facts (dates, numbers, names, events).
Skip opinions and general statements.

Text: {text}

Factual claims:""")]).content
    claims = [line.strip() for line in response.split('\n') if line.strip() and line.strip()[0].isdigit()]
    return claims

sample_text = """Python was created by Guido van Rossum in 1991. It is now the most popular programming language according to the TIOBE index. Python 3.12 was released in October 2023 and includes significant performance improvements. The language is used by companies including Google, NASA, and Instagram."""

claims = extract_claims(sample_text)
print(f"Extracted {len(claims)} claims:")
for c in claims: print(f"  {c}")

Build the Claim Verifier

def verify_claim(claim: str, evidence: str) -> dict:
    """Verify a claim against provided evidence."""
    response = llm([HumanMessage(content=f"""Verify this claim against the evidence provided.
Claim: {claim}
Evidence: {evidence}

Is the claim:
A) SUPPORTED by the evidence
B) CONTRADICTED by the evidence
C) NOT_ADDRESSED by the evidence (cannot verify)

Answer with just A, B, or C, then one sentence explanation.""")]).content
    verdict = "SUPPORTED" if "A" in response[:10] else "CONTRADICTED" if "B" in response[:10] else "UNVERIFIED"
    return {"claim": claim, "verdict": verdict, "explanation": response[5:].strip()[:200]}

Build the Full Detection Pipeline

def hallucination_detection_pipeline(agent_output: str, evidence_fn) -> dict:
    """Full pipeline: extract claims → get evidence → verify each claim."""
    print("Step 1: Extracting claims...")
    claims = extract_claims(agent_output)
    print(f"  Found {len(claims)} verifiable claims")
    results = []
    print("Step 2: Verifying claims...")
    for claim in claims:
        evidence = evidence_fn(claim)
        result = verify_claim(claim, evidence)
        results.append(result)
        icon = {"SUPPORTED":"✅","CONTRADICTED":"❌","UNVERIFIED":"⚠️"}.get(result["verdict"],"?")
        print(f"  {icon} {claim[:60]}...")
    supported = sum(1 for r in results if r["verdict"]=="SUPPORTED")
    contradicted = sum(1 for r in results if r["verdict"]=="CONTRADICTED")
    unverified = sum(1 for r in results if r["verdict"]=="UNVERIFIED")
    reliability = supported / len(results) if results else 0
    return {"total_claims": len(results), "supported": supported, "contradicted": contradicted,
            "unverified": unverified, "reliability_score": round(reliability, 2), "details": results}

def mock_evidence(claim): return "Python was created by Guido van Rossum and released in 1991. Python 3.12 was released in 2023."
report = hallucination_detection_pipeline(sample_text, mock_evidence)
print(f"\nReliability score: {report['reliability_score']:.0%}")
print(f"  Supported: {report['supported']} | Contradicted: {report['contradicted']} | Unverified: {report['unverified']}")
