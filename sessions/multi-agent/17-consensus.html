Society of Mind, LLM debate, majority voting, ranked choice, and Delphi method for multi-agent truth-seeking.

Society of Mind

Marvin Minsky's Society of Mind (1986) proposed that human intelligence arises not from a single monolithic process but from the interaction of many simple agents — each responsible for a narrow capability, collectively producing intelligent behaviour. This maps beautifully to LLM multi-agent systems: rather than asking one model to do everything, build a society of specialized agents that debate, critique, and synthesize.

Core idea: Multiple agents independently reason about the same problem, then combine their conclusions through a consensus mechanism. This reduces the impact of any single agent's bias or hallucination.

LLM Debate (Du et al., 2023)

Yilun Du and colleagues showed in 2023 that having multiple LLM agents debate each other significantly improves factual accuracy and reasoning quality. The debate protocol:

  1. Round 1: Each agent independently answers the question.
  2. Round 2+: Each agent reads all other agents' answers and writes a critique + revised answer.
  3. Final round: Agents produce their final position after N rounds of debate.
  4. Synthesis: A judge agent or voting mechanism selects the best final answer.

The paper showed debate improved performance on arithmetic, reasoning, and factual tasks — outperforming single-agent chain-of-thought on complex problems.

Delphi Method for Agents

The Delphi method is a structured expert consensus technique from the 1950s (RAND Corporation) that iteratively converges expert opinion. Adapted for LLM agents:

from langchain_openai import ChatOpenAI
from langchain.schema import HumanMessage, SystemMessage

llm = ChatOpenAI(model="gpt-4o-mini", temperature=0.7)

AGENT_PERSONAS = [
    ("Optimist", "You are an optimistic analyst who focuses on opportunities and potential."),
    ("Skeptic",  "You are a skeptical analyst who focuses on risks, uncertainties, and counter-evidence."),
    ("Pragmatist","You are a pragmatic analyst who focuses on practical implementation and real-world constraints."),
]

def delphi_round(question: str, previous_responses: list = None, round_num: int = 1) -> dict:
    """Run one Delphi round: each agent answers, optionally seeing previous round answers."""
    responses = {}
    context = ""
    if previous_responses:
        context = "\n\nPrevious round answers from other analysts:\n" + "\n".join(
            [f"- {name}: {resp[:200]}" for name, resp in previous_responses.items()]
        ) + "\n\nConsider these perspectives in your answer, but maintain independent judgment."

    for name, system_prompt in AGENT_PERSONAS:
        response = llm([
            SystemMessage(content=system_prompt),
            HumanMessage(content=f"Round {round_num}. Question: {question}{context}\n\nYour analysis (2-3 sentences):")
        ]).content
        responses[name] = response
        print(f"  [{name}]: {response[:120]}...")
    return responses

def run_delphi(question: str, rounds: int = 2) -> str:
    print(f"Question: {question}\n")
    all_responses = None
    for r in range(1, rounds+1):
        print(f"=== Round {r} ===")
        all_responses = delphi_round(question, all_responses, r)
    # Synthesize
    synthesis_prompt = f"Synthesise these expert analyses into one balanced conclusion:\n" + \
        "\n".join([f"{k}: {v}" for k,v in all_responses.items()])
    final = llm([HumanMessage(content=synthesis_prompt)]).content
    return final

Self-Consistency Voting

Self-consistency (Wang et al., 2022) is a simpler consensus technique: sample the same LLM multiple times with temperature > 0, then take majority vote. This works because different reasoning paths often converge on the correct answer even when individual paths vary.

from collections import Counter

def self_consistency_vote(question: str, n_samples: int = 5) -> str:
    """Sample LLM n times and take majority vote on the final answer."""
    llm_sampler = ChatOpenAI(model="gpt-4o-mini", temperature=0.7)
    answers = []
    for i in range(n_samples):
        response = llm_sampler([HumanMessage(content=f"Answer with just the final answer (no explanation):\n{question}")]).content.strip()
        answers.append(response)
        print(f"  Sample {i+1}: {response}")
    # Vote
    counts = Counter(answers)
    winner, votes = counts.most_common(1)[0]
    confidence = votes / n_samples
    print(f"\nVote result: '{winner}' ({votes}/{n_samples} = {confidence:.0%} confidence)")
    return winner
Cost vs quality tradeoff: Debate and Delphi multiply LLM calls (3 agents × 2 rounds × synthesis = 7+ calls). Self-consistency is cheaper (n samples of one model). Use debate for high-stakes fact-checking; self-consistency for numerical estimation; majority vote for MCQ-style tasks.

import os; os.environ["OPENAI_API_KEY"] = "your-key"
from langchain_openai import ChatOpenAI
from langchain.schema import HumanMessage, SystemMessage
from collections import Counter

llm = ChatOpenAI(model="gpt-4o-mini", temperature=0.3)

agents = {
    "FactChecker": "You are a rigorous fact-checker. Evaluate claims against known evidence. Be specific about what is confirmed, uncertain, or false.",
    "Advocate":    "You are a devil's advocate. Find ways the claim could be true even if it seems false. Look for nuance.",
    "Skeptic":     "You are deeply skeptical. Find every reason the claim might be false, exaggerated, or misleading.",
}
claim = "The Great Wall of China is visible from space with the naked eye."
print(f"Claim to fact-check: '{claim}'")
2

Round 1: Independent Positions

def agent_respond(name, system, claim, context=""):
    return llm([
        SystemMessage(content=system),
        HumanMessage(content=f"Claim: '{claim}'\n{context}\nYour analysis (3 sentences max):")
    ]).content

round1 = {}
print("=== ROUND 1: Independent Positions ===")
for name, system in agents.items():
    r = agent_respond(name, system, claim)
    round1[name] = r
    print(f"\n[{name}]: {r}")
3

Round 2: Debate After Seeing Others

round2 = {}
print("=== ROUND 2: Revised Positions After Debate ===")
for name, system in agents.items():
    others = "\n".join([f"- {n}: {r[:150]}" for n,r in round1.items() if n != name])
    context = f"\nOther analysts said:\n{others}\nRevise your position if warranted:"
    r = agent_respond(name, system, claim, context)
    round2[name] = r
    print(f"\n[{name}]: {r}")
4

Verdict and Consensus

print("=== FINAL VERDICT ===")
# Ask each agent for a binary verdict
verdicts = {}
for name, system in agents.items():
    v = llm([
        SystemMessage(content=system),
        HumanMessage(content=f"Based on our debate, is this claim TRUE or FALSE? Answer with just TRUE or FALSE.\nClaim: '{claim}'\nYour round 2 position: {round2[name][:200]}")
    ]).content.strip().upper()
    verdict = "TRUE" if "TRUE" in v else "FALSE"
    verdicts[name] = verdict
    print(f"  {name}: {verdict}")

counts = Counter(verdicts.values())
final_verdict = counts.most_common(1)[0][0]
confidence = counts.most_common(1)[0][1] / len(agents)
print(f"\nConsensus Verdict: {final_verdict} ({confidence:.0%} agreement)")

# Synthesize final explanation
synth = llm([HumanMessage(content=f"The claim '{claim}' was debated by 3 analysts who concluded {final_verdict}. Write a 2-sentence explanation of the verdict based on:\n" + "\n".join([f"{n}: {r}" for n,r in round2.items()]))]).content
print(f"\nExplanation: {synth}")
